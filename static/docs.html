.scopecam PNG-Format

ScopeCam streamt den Kameraeingang auf eine Zeichenfläche. Darauf können Linien, Pfeile, Bemaßungen, Texte, Rechtecke und mehr eingezeichnet werden. Projekte werden als .scopecam-Dateien gespeichert (gültiges PNG mit eingebetteten Metadaten).

Werkzeuge
⬡
Auswahl
Objekte auswählen, verschieben, skalieren, drehen
S
›

Das Standard-Werkzeug. Klicke ein Objekt an um es auszuwählen. Ziehe eine Rubber-Band-Box um mehrere Objekte gleichzeitig zu erfassen.

  • Klick auf Objekt → auswählen
  • Klick+Drag im Leeren → Mehrfachauswahl per Rahmen
  • Handles an den Ecken → Skalieren (Shift = gleichmäßig)
  • Runder Handle oben → Drehen
  • Entf → ausgewähltes Objekt löschen
Tipp: Alt+X / Alt+Y sperrt die Bewegung auf eine Achse. Ctrl während dem Verschieben deaktiviert das Einrasten.
✋
Hand / Pan
Ansicht verschieben ohne Objekte zu bewegen
H
›

Klicken und ziehen bewegt die gesamte Ansicht (Pan). Das Werkzeug deaktiviert das Auswählen von Objekten.

  • Werkzeug aktivieren mit H oder Klick auf Hand-Button
  • Klicken und ziehen → Ansicht schieben
  • Mausrad → Zoomen
Tipp: Leertaste gedrückt halten aktiviert die Hand temporär — loslassen kehrt zum vorherigen Werkzeug zurück.
Zeichenwerkzeuge
╱
Linie
Einfache gerade Linie
L
›
  • Werkzeug mit L aktivieren
  • Klicken und ziehen → Linie aufspannen
  • Loslassen → Linie fertig
Winkeleinrasten: Shift gedrückt halten → Linie rastet auf 0°, 45°, 90°, 135° ein.
→
Pfeil
Linie mit Pfeilspitze am Ende
A
›

Genau wie Linie, aber das Ende wird mit einer Pfeilspitze versehen (Dreieck, gefüllt mit der Strichfarbe).

  • A drücken
  • Startpunkt klicken und zur Zielposition ziehen
  • Loslassen → Pfeil wird erstellt
Tipp: Pfeil + Länge = Bemaßung? Benutze stattdessen das Bemaßungs-Werkzeug D.
⟷
Bemaßung
Maßlinie mit automatischem Längen-Label
D
›

Erstellt eine Maßlinie mit Pfeilspitzen an beiden Enden und einem zentrierten Längen-Label (px oder mm wenn kalibriert).

  • D drücken
  • Startpunkt ziehen zur Endposition
  • Label zeigt automatisch px-Länge bzw. mm nach Kalibrierung
Label überschreiben: Objekt auswählen → im Eigenschaften-Panel ✓ "Maßung überschreiben" → eigenen Text eingeben.
Kalibrierung: Werkzeug-Leiste → Kalibrierung → Linie über bekannte Strecke ziehen → mm eingeben.

Rechteck
Gefülltes oder transparentes Rechteck
R
›
  • R drücken
  • Ecke klicken und zur gegenüberliegenden Ecke ziehen
  • Loslassen → Rechteck erstellt
Quadrat: Shift gedrückt halten → quadratisches Seitenverhältnis.
◯
Kreis / Ellipse
Kreis oder ovale Form
C
›
  • C drücken
  • Mittelpunkt klicken und ziehen
  • Loslassen → Kreis erstellt
Tipp: Farbe und Strichbreite im Eigenschaften-Panel oder der Werkzeugleiste einstellen.
T
Text
Editierbares Textfeld auf dem Canvas
T
›
  • T drücken
  • Klick auf die gewünschte Position
  • Text tippen → Cursor erscheint, direkt editierbar
  • Esc oder Klick außerhalb → Textbearbeitung beenden
Nacherbeiten: Doppelklick auf bestehenden Text → Bearbeitungsmodus. Im Eigenschaften-Panel: Schriftart, Größe, B/I/U, Farbe.
✏
Freihand
Freihandzeichnen wie mit Stift
F
›
  • F drücken
  • Klicken und ziehen → Linie folgt der Mausbewegung
  • Loslassen → Pfad wird als Objekt hinzugefügt
Strichbreite und Farbe werden aus dem Eigenschaften-Panel übernommen.
⌇
Polylinie
Mehrere verbundene Liniensegmente
P
›

Ideal für PCB-Leiterbahnen oder Umrisse mit mehreren geraden Segmenten. Alle Segmente erhalten automatisch dieselbe linkGroup.

  • P drücken
  • Ersten Punkt klicken
  • Weitere Punkte klicken — Vorschau-Linie folgt der Maus
  • Doppelklick oder Enter → Polylinie abschließen
  • Esc → Abbrechen ohne zu speichern
Winkeleinrasten: Shift gedrückt → 45°-Raster aktiv.
💬
Callout
Hinweislinie mit Textfeld (Beschriftung)
—
›

Erstellt eine Linie vom Ankerpunkt zum Textfeld — gut für Bauteilbeschriftungen.

  • Callout-Werkzeug in der Werkzeugleiste aktivieren
  • Erster Klick = Ankerpunkt (zeigt auf das Bauteil)
  • Zweiter Klick = Textposition
  • Text eingeben → Esc oder Klick außerhalb fertigstellen
Linie und Text werden automatisch als Gruppe verknüpft (linkGroup).
📏
Schnellmessung
Distanz zwischen zwei Punkten messen (ohne Objekt zu erstellen)
—
›
  • Schnellmessung in der Werkzeugleiste aktivieren
  • Ersten Punkt klicken
  • Zweiten Punkt klicken → Distanz wird als Overlay neben dem Cursor angezeigt
  • Ergebnis: px oder mm (wenn kalibriert)
Kein Objekt wird erstellt — nur Anzeige. Für permanente Maße das Bemaßungs-Werkzeug verwenden.
⚖
Kalibrierung
px/mm-Verhältnis aus bekannter Strecke berechnen
—
›
  • Kalibrierung-Werkzeug klicken
  • Linie über eine bekannte Strecke auf dem Bild ziehen (z.B. 10 mm zwischen zwei Pads)
  • Länge in mm eingeben → Bestätigen
  • Ab jetzt zeigen alle Bemaßungen Werte in mm an
Kalibrierung gilt nur für den aktuellen Zoom-Level des Kamerabildes. Bei Zoom-Änderung erneut kalibrieren.

Verknüpfen / Gruppieren
Objekte zu einer logischen Gruppe verbinden
Ctrl+G
›

Verknüpfte Objekte teilen eine linkGroup-UUID. Sie bleiben einzeln selektierbar und editierbar, lassen sich aber gemeinsam verschieben wenn alle ausgewählt sind.

  • Mindestens 2 Objekte auswählen

  • Ctrl+Shift+G → Verknüpfung aufheben

Batch-Umbenennen

—
›

  • Bearbeiten → Batch-Umbenennen (oder Button in Objekt-Manager)
  • Muster eingeben und Vorschau prüfen
  • OK → alle Objekte werden umbenannt

Verfügbare Platzhalter:

%d
Laufende Nummer (1, 2, 3…)
%D
Buchstabe (A, B, C…)
%n
Alter Name des Objekts
Beispiel: Muster C%d → C1, C2, C3 …

Tab-System
Mehrere Canvas-Ansichten gleichzeitig verwalten
—
›

Jeder Tab hat einen eigenen Canvas, eigene History, eigene Ebenen und einen eigenen Speicherpfad.

  • Tab-Leiste (oben): + Button → neuer Tab
  • Klick auf Tab-Name → wechseln
  • ✕-Button → Tab schließen (ungespeicherte Änderungen → Dialog)
  • Letzter Tab geschlossen → neuer Kamera-Tab wird automatisch erstellt
Blauer Punkt im Tab-Header = ungespeicherte Änderungen. Browser-Schließen warnt ebenfalls.

—
›

Der Live-Stream wird eingefroren — das letzte Frame bleibt sichtbar. Zeichnen ist weiterhin möglich. Der WebSocket bleibt aktiv.

  • Capture-Panel → Einfrieren-Button (oder Konsole: freeze on)
  • Button wird blau → Stream eingefroren
  • Nochmals klicken → Stream läuft wieder

Raster
Sichtbares Gitter mit optionalem Einrasten
Ctrl+'

  • Ansicht → Raster einrasten → Snap an Gitterpunkten
  • Raster-Einstellungen: Größe, Farbe, Deckkraft, Ursprung

Horizontale und vertikale Referenzlinien
—
›

  • Aus dem vertikalen Lineal ziehen → vertikale Hilfslinie
  • Erster Klick auf Hilfslinie → auswählen (gelb)
  • Klick + Drag → verschieben
  • Über den Canvas-Rand ziehen → löschen

🧲
Objekt-Einrasten
Magnetisch an Ecken und Mittelpunkten anderer Objekte
—
›

Beim Verschieben rastet ein Objekt automatisch an Ecken, Kantenmittelpunkten und Endpunkten von Linien anderer Objekte ein (Threshold: 12px). Ein cyan-farbiger Kreis zeigt den Snap-Punkt an.

Ctrl gedrückt halten → Snap temporär deaktivieren. Achssperrung (Alt+X/Alt+Y) deaktiviert Snap ebenfalls.
📐
PCB-Kanten-Snap (Beta)
Objekte rasten an Bildkanten des PCB-Fotos ein
—
›

Analysiert beim Verschieben eines Objekts den Bildsensor entlang der Objektkanten und rastet am stärksten Gradienten ein.

  • Ansicht → PCB-Kanten → Aktivieren
  • Suchradius und Gradient-Schwellwert anpassen
  • Objekte verschieben → Snap an PCB-Strukturen
Feature befindet sich im Beta-Stadium. Funktioniert am besten mit hohem Kontrast zwischen Bauteil und Hintergrund.

💬
KI-Chat & Aktionen
KI sieht Canvas + Live-Frame und kann Objekte erstellen
—
›

Das KI-Panel (linke Seitenleiste) zeigt den Chat. Mit jedem Request sendet die KI automatisch:

  • Aktuellen Canvas-Zustand (alle Objekte, Ebenen, Hilfslinien)
  • Aktuelles Live-Frame (JPEG)
  • Deine Nachricht

Die KI kann folgende Aktionen ausführen:

Erstellen
rect, circle, line, arrow, dimension, text
Bearbeiten
move, delete, setProps, rename
Strukturieren
link, unlink, createLayer, moveToLayer
Hilfslinien
addGuide, moveGuide, removeGuide, clearGuides
Bereich-Auswahl (⊠): Button im KI-Panel → Rubber-Band auf Canvas → nur dieser Ausschnitt wird gesendet. Koordinaten-Offset wird automatisch verrechnet.
Berechtigungen: Einstellungen → KI → Berechtigungen → granular steuern was die KI darf.
